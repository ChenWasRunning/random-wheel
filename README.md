# Random Wheel

Static GitHub Pages site for the blind box random wheel.

Published URL after GitHub Pages is enabled:

https://ChenWasRunning.github.io/random-wheel/

## Deploy

1. Create an empty public GitHub repository named `random-wheel` under `ChenWasRunning`.
2. Push this local repository to GitHub:

   ```sh
   git push -u origin main
   ```

3. In GitHub, open `Settings > Pages`, set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

The page entrypoint is `index.html`.
